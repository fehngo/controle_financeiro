# 💰 Sistema de Controle Financeiro

Aplicação desenvolvida em Python para gerenciamento de finanças pessoais, permitindo registrar entradas e saídas, visualizar saldo e organizar transações.

---

## 🚀 Funcionalidades

* ✅ Adicionar transações (entrada e saída)
* 📋 Listar transações
* 💰 Calcular saldo automaticamente
* 🔍 Filtrar por tipo (entrada ou saída)
* ❌ Excluir transações

---

## 🧠 Conceitos aplicados

* CRUD (Create, Read, Update, Delete)
* Manipulação de arquivos JSON
* Estruturas de dados (listas e dicionários)
* Lógica de negócio (cálculo de saldo)
* Organização de código em módulos

## ⚙️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/controle_financeiro.git
```

2. Acesse a pasta:

```
cd controle_financeiro
```

3. Execute o programa:

```
python main.py
```
